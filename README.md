Linux Story 是一个全新风格的 Linux 资讯网站，每天为您带来第一手的开源资讯和新鲜温暖的科技故事。开源有意思，除了热气腾腾的开源故事，我们也想说说“情话”，讲讲“情怀”，我们是Linux Story，我们分享故事，更想和你一起创造故事。
